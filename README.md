# 🃏 Memory Game

A classic card flip memory matching game with emoji cards.

## Features
- 3 difficulty levels: Easy (4×4), Medium (5×4), Hard (6×4)
- Move counter and live timer
- Matched pairs tracker
- Smooth 3D card flip animation
- Win screen with time and move summary
- Restart anytime

## Built With
- HTML5
- CSS3 (3D transforms)
- Vanilla JavaScript

## How to Run
Clone the repo and open `index.html` in your browser.
